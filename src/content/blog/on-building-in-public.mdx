I've been a software engineer for a while now, and I've noticed a pattern: I read a lot, bookmark a lot, and produce very little in the way of writing.

This site is an attempt to fix that.

## what cyber-rhizome means

A rhizome, in botany, is a root system that grows horizontally — no center, no hierarchy, just nodes connected to other nodes. Ginger is a rhizome. So is grass. The philosopher Deleuze used it as a metaphor for non-linear thinking: knowledge that spreads and connects rather than organizes into trees.

I like that. This site is going to be a mix of things — software engineering, fermentation projects, sewing, whatever I'm thinking about. The connective tissue is the tags, not the categories.

## what I'm working on

At the moment my main project is **sentinel-l7**, an AI-driven compliance monitoring platform for financial transactions. It's built on Laravel, Vue 3, and Upstash Redis Streams, and it's been a genuinely interesting engineering problem.

I'll write more about that soon.

## what this site won't be

It won't be a polished content marketing funnel. It won't have a newsletter pop-up. It's just writing, for the practice of it, with a comment section powered by GitHub Discussions in case anyone wants to talk about it.

If you're reading this: hello.
